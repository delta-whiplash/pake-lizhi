# pake-lizhi — AUR Package

AUR package for Lizhi wrapped as an AppImage using [Pake](https://github.com/tw93/Pake).

## Install

```bash
paru -S pake-lizhi
yay  -S pake-lizhi
```

## Runtime dependency

Requires **FUSE 2**:

```bash
sudo pacman -S fuse2
```

## Automated updates

Daily GitHub Actions workflow checks for new Pake releases and auto-updates this package.
